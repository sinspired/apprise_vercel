# Ntfy

⭐⭐⭐⭐⭐

> [ `iOS` + `Android` + `Web` ] 设置极其简单，仅需设置 主题（Topic）即可，但和 [Bark (iOS)](Bark) 相比可选参数较少

> **Ntfy** 是一个开源的推送服务，支持通过主题订阅在 iOS、Android 和 Web 上接收通知。[ⓘ 查看详细教程](https://appriseit.com/services/ntfy)

## 📦 URL 格式

`ntfy://{topic}`  

## ⚡️ 快速步骤

1. 打开 [Ntfy 官网](https://ntfy.sh)，点击 `Open app` -> `Subscribe to topic`
2. 手机下载应用 `订阅主题`
3. 记录 `topic`
4. 按格式拼接目标 `URL`，Enjoy! 🎉

### 示例

```bash
ntfy://mytopic
ntfy://user:pass@ntfy.sh/mytopic
```
