# Discord

> [!TIP]
> **Discord** 支持通过 Webhook 向频道发送通知消息。[ⓘ 查看详细教程](https://appriseit.com/services/discord)

## 📦 URL 格式

`discord://{webhook_id}/{webhook_token}`

## ⚡️ 快速步骤

1. 打开 Discord，进入目标服务器频道设置  
2. 创建一个新的 **Webhook**，复制 `{webhook_id}` 与 `{webhook_token}`  
3. 按格式拼接目标 `URL`，Enjoy! 🎉

### 示例

```bash
discord://123456789012345678/abcdefghijklmnopqrstuvwxyz
```
