# Line

> [!TIP]
> **Line** 支持通过 Messaging API 向用户或群组发送通知。[ⓘ 查看详细教程](https://appriseit.com/services/line)

## 📦 URL 格式

`line://{access_token}`

## ⚡️ 快速步骤

1. 登录 [Line Developers](https://developers.line.biz)，创建 Messaging API Channel  
2. 获取 `{access_token}`  
3. 按格式拼接目标 `URL`，Enjoy! 🎉

### 示例

```bash
line://EAA1234567890abcdefghijklmnopqrstuvwx
```
