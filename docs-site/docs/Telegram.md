# Telegram

> [!TIP]
> 通过 Telegram Bot 向自己、频道、群组发送通知消息。[ⓘ 查看详细教程](https://appriseit.com/services/telegram)

## 📦 URL 格式

`tgram://{bot_token}/{chat_id}`
`tgram://{bot_token}/{chat_id1}/{chat_id2}/{chat_id3}/`

## ⚡️ 快速步骤

1. 找 `@BotFather` 创建机器人，获取 `{bot_token}`
2. 搜索 `User Info • Get ID • IDbot` ，选择 `@userinfobot`，获取 `{chat_id}`
3. 按格式拼接目标 `URL`，Enjoy! 🎉

> [!NOTE]
> 如需向群组、频道发送通知，需将你创建的机器人添加到相应群组或频道。可通过 `https://api.telegram.org/bot{你的机器人token}/getUpdates` 获取群组和频道的 `{chat_id}`
