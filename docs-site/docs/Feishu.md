# Feishu (飞书)

> [!TIP]
> **飞书** 支持通过机器人 Webhook 向群组发送通知。[ⓘ 查看详细教程](https://appriseit.com/services/feishu)

## 📦 URL 格式

`feishu://{app_id}/{app_secret}/{chat_id}`

## ⚡️ 快速步骤

1. 登录飞书开放平台，创建一个应用并启用机器人功能  
2. 获取 `{app_id}` 与 `{app_secret}`  
3. 将机器人加入目标群组，获取 `{chat_id}`  
4. 按格式拼接目标 `URL`，Enjoy! 🎉

### 示例

```bash
feishu://cli_9f8a1234567890ab/csk_abcdef1234567890/oc_abcdef1234567890
```
