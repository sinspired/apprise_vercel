# Slack

> [!TIP]
> **Slack** 支持通过 Webhook 或 Bot Token 向频道发送通知。[ⓘ 查看详细教程](https://appriseit.com/services/slack)

## 📦 URL 格式

`slack://{token}/{channel}`

## ⚡️ 快速步骤

1. 打开 Slack API 页面，创建一个应用并启用 **Incoming Webhooks**  
2. 获取 `{token}` 并指定目标 `{channel}`  
3. 按格式拼接目标 `URL`，Enjoy! 🎉

### 示例

```bash
slack://xoxb-1234567890-abcdefghijklmnopqrst/notifications
```
