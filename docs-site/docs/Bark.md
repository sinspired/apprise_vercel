# Bark

⭐⭐⭐⭐⭐

> [ `iOS` 首推 ] 因其个性化程度最高，下载和使用方便。安卓建议使用 [Ntfy](Ntfy)

> **Bark** 是一款免费的 iOS 推送工具，支持推送分组、自定义图标与铃声、时效性通知、重要警告等高级特性。[ⓘ 查看详细教程](https://appriseit.com/services/bark)

## 📦 URL 格式

`bark://{host}/{device_key}`

## ⚡️ 快速步骤

1. AppStore 下载 `Bark`
2. 打开软件 -> 你会看到类似 `https://api.day.app/j300012fl9y0b5AW9g9Nsejb8P/这里改成你自己的推送内容`
   - `{device_key}` = `j300012fl9y0b5AW9g9Nsejb8P`
   - `{host}` = `api.day.app`
3. 按格式拼接 `URL`，Enjoy! 🎉

### 示例

```bash
bark://api.day.app/j300012fl9y0b5AW9g9Nsejb8P
```
