# DingTalk (钉钉)

> [!TIP]
> **钉钉** 支持通过机器人 Webhook 向群组发送通知。[ⓘ 查看详细教程](https://appriseit.com/services/dingtalk)

## 📦 URL 格式

`dingtalk://{access_token}`
`dingtalk://{ApiKey}/{ToPhoneNo}`

## ⚡️ 快速步骤

1. 打开钉钉群组设置，添加一个自定义机器人  
2. 获取 `{access_token}`  
3. 按格式拼接目标 `URL`，Enjoy! 🎉

### 示例

```bash
dingtalk://abcdef1234567890abcdef1234567890
```
